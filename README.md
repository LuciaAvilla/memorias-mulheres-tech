Sistema de cadastro e consulta desenvolvido como Desafio Final do curso de Python da WoMakersCode, com foco em preservar e dar visibilidade às histórias de mulheres que transformaram a tecnologia.

---

## Sobre

Este projeto tem como objetivo criar um sistema capaz de registrar, consultar e organizar informações sobre mulheres que marcaram a história da tecnologia.

Além disso, o projeto reforça conceitos importantes de lógica de programação e manipulação de dados em Python.

---

## Funcionalidades

O sistema oferece as seguintes opções:

**Cadastrar Mulher Tech**  
  Registro de nome, área de atuação, contribuição e período histórico  

**Listar Histórico**  
  Exibição de todas as mulheres cadastradas no sistema  

**Buscar por Nome**  
  Filtro inteligente para localizar uma pioneira específica  

**Exibir Estatística**  
  Resumo com o total de registros e pioneiras de diferentes épocas  

**Persistência de Dados**  
  Salvamento automático em arquivo `.txt`, garantindo que os dados não se percam  

---

## Como Rodar

1. Certifique-se de ter o **Python 3** instalado.
2. Clone o repositório:
   git clone [https://github.com/LuciaAvilla/memorias-mulheres-tech.git](https://github.com/LuciaAvilla/memorias-mulheres-tech.git)
3. Execute o script: `python desafio_mulheres.py`
